### Step 3: Convert MP3 to OGG
**Why OGG?**
- 30% smaller than MP3
- Better quality at same bitrate
- Godot native support

**Online Method (Easiest):**
1. Visit: https://convertio.co/mp3-ogg/
2. Upload MP3 files
3. Download OGG files

**FFmpeg Method (Batch):**
```bash
ffmpeg -i input.mp3 -c:a libvorbis -q:a 5 output.ogg
```
Quality 5 = ~128 kbps

**Audacity Method:**
1. Open MP3 in Audacity
2. File ‚Üí Export ‚Üí Export as OGG
3. Quality: 5

### Step 4: Import to Godot
1. Place all `.ogg` files in `res://audio/music/`
2. In Godot, select each file
3. Set Import settings:
   - **Loop:** ENABLED ‚úÖ
   - **Loop Offset:** 0
   - **Compression:** Vorbis
4. Reimport all files

## üö® TROUBLESHOOTING

**Problem:** Music is too loud/quiet
