# Sound Effects (SFX) Folder

This folder contains all 35+ sound effects for the game, organized by category.

## Folder Structure:

```
sfx/
├── combat/         - Combat sounds (9 SFX)
├── weapons/        - Weapon sounds (8 SFX)
├── ui/             - UI sounds (7 SFX)
├── bosses/         - Boss sounds (6 SFX)
└── environment/    - Environmental sounds (5 SFX)
```

## Required SFX Files (35 total):

### COMBAT (9 SFX):
- `hit_impact.ogg` - Weapon hits enemy
- `enemy_death.ogg` - Enemy dies
- `player_hurt.ogg` - Player takes damage
- `level_up.ogg` - Level up sound
- `xp_collect.ogg` - XP orb pickup
- `critical_hit.ogg` - Critical hit special
- `dodge_roll.ogg` - Dodge sound
- `parry_block.ogg` - Parry/block
- `enemy_spawn.ogg` - Enemy appears

### WEAPONS (8 SFX):
- `sword_slash.ogg` - Melee weapon swing
- `bow_shoot.ogg` - Arrow/projectile shoot
- `lightning_zap.ogg` - Electric attack
- `earth_slam.ogg` - Ground pound
- `daggers_slash.ogg` - Fast triple slash
- `acid_sizzle.ogg` - Acid/poison effect
- `flute_notes.ogg` - Musical weapon
- `magic_slash.ogg` - Kiku sword special

### UI (7 SFX):
- `button_click.ogg` - Button press
- `menu_open.ogg` - Menu appears
- `menu_close.ogg` - Menu closes
- `item_pickup.ogg` - Item collected
- `notification.ogg` - Alert ping
- `fragment_collect.ogg` - Special item
- `chat_message.ogg` - Chat notification

### BOSSES (6 SFX):
- `dragon_roar.ogg` - Fire Dragon roar
- `vampire_laugh.ogg` - Vampire Lord laugh
- `boss_phase_change.ogg` - Phase transition
- `pam_anime_shout.ogg` - Pam "NANI?!"

### ENVIRONMENT (5 SFX):
- `footstep_grass.ogg` - Walking on grass
- `footstep_sand.ogg` - Walking on sand
- `footstep_snow.ogg` - Walking on snow
- `cage_shatter.ogg` - Crystal cage breaks
- `ritual_chant.ogg` - Ambient chanting

## Format Requirements:
- **Format**: OGG Vorbis
- **Quality**: ~128 kbps
- **Duration**: Short (0.1 - 3 seconds for most)
- **Loop**: DISABLED (except ambient sounds like ritual_chant)
- **Compression**: Vorbis
