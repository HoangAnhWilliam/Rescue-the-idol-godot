1. Go to: https://freesound.org/
2. Click "Sign up" (free forever)
3. Verify email

### Step 2: Download Each Sound
1. Click each link above
2. Click green "Download" button
3. Save to folder: `downloaded_sfx/`
4. Repeat for all 35 sounds
5. **Time:** ~30-45 minutes

### Step 3: Rename Files
Rename downloaded files to match naming convention:
- `320181__dland__punch-impact.wav` ‚Üí `hit_impact.wav`
- `249819__spookymodem__monster-death.wav` ‚Üí `enemy_death.wav`
- etc.

### Step 4: Convert to OGG (if needed)
Some files may be WAV or MP3. Convert to OGG:

**Online:** https://convertio.co/
**Audacity:** File ‚Üí Export ‚Üí Export as OGG (Quality 5)

### Step 5: Import to Godot
1. Place files in correct folders:
   ```
   res://audio/sfx/combat/
   res://audio/sfx/weapons/
   res://audio/sfx/ui/
   res://audio/sfx/bosses/
   res://audio/sfx/environment/
   ```

2. Import settings for EACH file:
   - **Loop:** DISABLED ‚ùå (except ambient like ritual_chant)
   - **Compression:** Vorbis
